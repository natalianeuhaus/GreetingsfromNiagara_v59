NIAGARA INDUSTRIAL & RADIOLOGICAL HISTORY — WORKING MAP
Version 0.2

WHAT IS INCLUDED
- 41 mapped industrial and waste locations from the ArcGIS Web Map draft
- 8 Hooker-area working boundaries
- 409 Niagara-related source records indexed from the Excel workbook
- Site popups with up to six automatic site-specific source matches
- A searchable research-archive panel
- Satellite, street, and historical aerial backgrounds
- GeoJSON and JSON files for future editing

IMPORTANT REVIEW NOTE
The source links in site popups were matched automatically using explicit site names,
facility names, street names, or project names found in the workbook. They are research
candidates, not final citations. Verify the document, page, site location, claim, and
evidence status before publication.

HOW TO VIEW
Open index.html in a web browser while connected to the internet.

HOW TO PUBLISH WITH GITHUB PAGES
1. Create a GitHub repository.
2. Upload index.html and the data folder.
3. Open Settings > Pages.
4. Publish from the main branch and root folder.
